# Student Registration Portal — EduTrack

A multi-page student registration and profile system built with semantic HTML5 and vanilla CSS. Demonstrates form design, multi-section data collection, and a polished profile card UI.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Multi-section registration form (personal info, academic details, preferences) |
| `profile.html` | Student profile card displaying all registered information in a structured layout |
| `success.html` | Registration confirmation with assigned roll number and orientation details |
| `style.css` | Shared stylesheet for all three pages |

## Assets

| File | Usage |
|------|-------|
| `images/pic.jpg` | Student avatar displayed on the profile card |

## Features

- Visual step indicator showing registration progress
- Multi-section form with personal, academic, and preference fields
- Pill/chip-style radio and checkbox selectors for technology preferences
- Profile card with banner, avatar, stats row, and detail grid
- Active interest tags displayed on the profile
- Registration confirmation with roll number and orientation schedule
- Consistent teal colour scheme throughout

## Tech Stack

- HTML5 (semantic elements: `header`, `section`, `form`, `footer`)
- CSS3 (custom properties, flexbox, grid, transitions, `aspect-ratio`)
- No external frameworks or JavaScript dependencies

## How to Run

Open `index.html` in any modern web browser. All internal links are relative paths, so the site works correctly from any local directory.
