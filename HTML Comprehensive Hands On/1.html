Starter Exercise

Use this page to practice building a full HTML document from scratch.

Practice targets

  • Create headings from h1 to h6.
  • Build links with a and test target="_blank".
  • Add ordered and unordered lists.
  • Insert images with clear alt text.
  • Try a basic contact form with input, label, and button elements.

Starter template

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<title>Practice Page</title>
</head>
<body>
	<header>...</header>
	<main>...</main>
	<footer>...</footer>
</body>
</html>
