# AI Degree Landing Page

A multi-page university programme landing page for a **Bachelor's in Artificial Intelligence**. Built with semantic HTML5 and vanilla CSS — no frameworks or dependencies.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Main landing page with programme overview, curriculum, careers, fees, and contact sections |
| `apply.html` | Multi-section application form with personal, academic, and preference fields |
| `success.html` | Application confirmation page with next-steps guidance |
| `style.css` | Shared stylesheet for all three pages |

## Features

- Sticky navigation bar with smooth anchor links
- Hero section with call-to-action buttons
- Stats bar highlighting key programme metrics
- Responsive card grid layouts for programme highlights and career paths
- Four-year curriculum breakdown
- Fee and scholarship information table
- Multi-section application form with HTML5 validation
- Mobile-responsive design

## Tech Stack

- HTML5 (semantic elements: `nav`, `section`, `footer`, `form`)
- CSS3 (custom properties, flexbox, grid, transitions)
- No external frameworks or JavaScript dependencies

## How to Run

Open `index.html` in any modern web browser. No build step or server required.
