# Restaurant Website — Karahi House

A fully-linked, multi-page restaurant website for **Karahi House**, an authentic Pakistani cuisine restaurant. Built with semantic HTML5 and vanilla CSS — no frameworks or dependencies.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Homepage with hero image, feature highlights, featured dishes, and opening hours |
| `menu.html` | Full restaurant menu organized into Starters, Main Course, Breads, Drinks, and Desserts |
| `gallery.html` | Photo gallery grid with hover overlay effects |
| `contact.html` | Contact details, opening hours, and a table reservation / feedback form |
| `style.css` | Shared stylesheet for all pages |

## Assets

All images are in the `images/` folder:

| File | Usage |
|------|-------|
| `black.jpeg` | Navigation logo |
| `krahi.jpg` | Hero section background |
| `1.jpeg – 6.jpeg` | Dish cards and gallery grid |
| `bg.jpeg` | Background asset |

## Features

- Sticky navigation bar with active-page highlighting
- Full-bleed hero with image overlay and call-to-action buttons
- Responsive card grid for features and dishes
- Hover zoom effects on gallery images
- Interactive star rating widget in the feedback form
- Consistent maroon and gold colour scheme throughout
- Multi-section footer with quick links

## Tech Stack

- HTML5 (semantic elements: `nav`, `section`, `footer`, `form`)
- CSS3 (custom properties, flexbox, grid, transitions, `object-fit`)
- No external frameworks or JavaScript dependencies

## How to Run

Open `index.html` in any modern web browser. All internal links are relative paths, so the site works correctly from any local directory.
